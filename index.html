About

Shopper is a really simple android application. It is just a list to manage current items you have to buy. Indeed, everyone can use it its own way.

The aim of the project is to keep it really simple. The principal functionality is sharing, because it is really useful :-)

Application can be found on Google Play here.

Project is host on GitHub

You will find here some information about the application development.
